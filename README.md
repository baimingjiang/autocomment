## Features

Insert head comments for you code file.

Ctrl+I: Insert comments in file head firstly.

```lua
--==============================--
    --@file: Context.lua
    --@desc: 
    --@time: 2019-5-16 11:55:11
    --@autor: autoheadcomment
    --@return Context
--==============================--

```

Ctrl+M: Insert modify author.

```lua
--==============================--
    --@file: Context.lua
    --@desc: 
    --@time: 2019-5-16 11:55:11
    --@autor: autoheadcomment
    --@modify: autoheadcomment at 2019-5-16 11:55:25
    --@return Context
--==============================--

```
